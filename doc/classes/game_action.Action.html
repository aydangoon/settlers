Class Action

An action is a request to change the game's state in some way such as by rolling the die, accepting a trade offer, playing a development card, etc.

All actions are verifiable, meaning they can (and always are) checked to be valid actions given the current game state. i.e. you can't roll twice on a turn.

Actions link between turn states in the turn state finite state machine.

A verified action can be applied to game state to update it in a safe and predictable manner.
